# afl-querier
Web app that allows for querying PIC-SURE using AFL query strings

# Pre-reqs:
- A valid `BD2K PICSURE API Key`
- üêç`python`
- `pip`
- `virtualenv-wrapper` (optional but reccomended)

# Installation:
- `mkvirtualenv afl-querier && workon afl-querier` (optional)
- `pip install -r requirements.txt`
- tweak values in `constants.py` to fit your needs
- `python app.py`
You should be able to view the site @ http://localhost:5000
